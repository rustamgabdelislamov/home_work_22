# Приложение интернет-магазина

Это приложение интернет-магазина, созданное с использованием Django. 
Оно включает в себя основные функции для управления товарами.

## Описание

Приложение позволяет пользователям просматривать товары, оформлять заказы. 

## Установка

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/rustamgabdelislamov/home_work_22.git
   ```

2. **Активируйте виртуальное окружение**:
   - **Windows**:
     ```bash
     poetry shell
     ```

3. **Установите зависимости**:
   ```bash
   poetry install
   ```

4**Примените миграции**:
   ```bash
   python manage.py migrate
   ```

5**Запустите сервер**:
   ```bash
   python manage.py runserver
   ```
